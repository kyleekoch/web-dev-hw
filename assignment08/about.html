My Instructable Choice

I chose this instructable because it is actually something that I love doing.
I find pour painting to be very fun because no painting ever comes out the same as the last.
It is a fun way to express myself and it is very easy to do. I also like to create these paintings because it is an easy way to liven up a room and add a pop of color.
If you want to learn more I recommend looking up tutorials on youtube, that is where I learned to do it,
and also the instructions that I provided give a great explanation for getting started.
